## Uso/Exemplos

```tmux

-- Local
CTRL + s - <Prefix>

-- Remoto
CTRL + a - <Prefix>

-- Janela
Prefix + c - Cria nova janela
Prefix + x - Fecha janela selecionada
Prefix + r - Renomeia a janela selecionada
Prefix + d - Desconecta a seccao 
Prefix + a - Vai para proxima janela 
Prefix + n - Muda numero da janela
Prefix + 1-9 - Seleciona janela por numero
Prefix + w - Lista janelas
Prefix + q - Renomeia seccao 
Prefix + : - Abre prompt de comando
Prefix + p - Reload de configuracoes

Shift + seta-esquerda - Move para janela a esquerda (Local)
Shift + seta-direita - Move para janela a direita (Local)
Shift + a + seta-esquerda - Move para janela em esquerda (Remoto)
Shift + a + seta-direita - Move para janela em direita (Remoto)

-- Painel
Prefix + CTRL + v - Cria painel vertical
Prefix + CTRL + b - Cria painel horizontal
Prefix + CTRL + x - Fecha painel selecionado
Prefix + CTRL + ! - Transforma painel em janela

Prefix + CTRL + seta-esquerda - Move para painel a esquerda
Prefix + CTRL + seta-direita - Move para painel a direita
Prefix + CTRL + seta-cima - Move para painel em cima
Prefix + CTRL + seta-baixo - Move para painel em baixo

```
